1️⃣ Identify the main axis in each case • flex-direction: row → main axis is horizontal • flex-direction: column → main axis is vertical ⸻ 2️⃣ Which margin directions become dominant? Margins applied along the main axis control spacing between items more noticeably. When direction changes, the same margin property can suddenly affect layout differently. ⸻ 3️⃣ How misunderstanding axes causes layout bugs Developers often add margins assuming left-right behavior, but when flex direction changes, margins act on a different axis. This causes unexpected spacing, misalignment, and broken responsive layouts.
