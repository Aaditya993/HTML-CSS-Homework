Solved: Flexbox Centers Correctly

Centered

When I tried to vertically center the box using margin, it didn’t work. The box stayed stuck at the top of the container, no matter how I adjusted the margins. This happens because margins only push elements within the normal document flow and don’t calculate the remaining space in the container correctly. When I switched to flexbox and set align-items: center, the box instantly centered vertically. Flexbox works because it understands the container as a whole and distributes free space evenly, so the element aligns exactly in the middle. This shows that while margins are good for spacing, they are unreliable for alignment, and using flexbox gives a predictable and consistent layout.
